Update currency-exchange prices from poe.ninja (browser)

Some currency-exchange items (the Verisium alloys, chase orbs) can't be priced from GGG's API — its standing offers are placeholder junk. Their real value lives on poe.ninja's currency exchange, which blocks our server. So pull it from your browser: open poe.ninja → PoE2 → your league → Currency, press F12 → Console, click Copy snippet, paste it there and hit Enter. It copies the prices — paste them below and Save. (Until you do, these show baked-in values.)

Jewel Pricer

Paste 1+ rare jewels — instant keep/junk triage, then an opt-in live floor check.

Stash search regex — find keepers

Paste into the in-game stash search (Ctrl+F). Each highlights jewels with a chase combo — quoted blocks are AND, | is OR. Single mods aren't here on purpose: almost every jewel has one, so they don't filter.

Triage is offline (zero trade calls) — a whole stash sorts instantly. Price check runs ONE rate-limited Trade2 search that ANDs the jewel's top mods, so the floor is "cheapest good jewel like this", not the junk floor. Rare jewels only; uniques (Timeless / Megalomaniac / Cluster) are a later pass.

Tab Tracker

Values the currency parked in a stash tab at live market prices and ranks it by what's worth selling — items with live buyers sit on top with their real sell value; ones with no buyers drop to the bottom as no live buyers. Make the tab public and give every item a price note in divine so the tool can read it — no login needed. Each lookup only sees the first 100 items, so spread a big tab across a few different price notes (e.g. 11, 12, 13, 14).
